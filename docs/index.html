Why use Recomendarr?

Stop digging through endless lists. Let your actual viewing habits drive automated content discovery.

AI Context Engine

Connects to OpenAI (or any compatible LLM) to analyze your history and explain *exactly* why a movie fits your mood.

Zero-Config Setup

No confusing `.env` files. Spin up via Docker and follow a beautiful 4-step UI wizard to connect everything instantly.

Direct Arr Injection

Automatically sends approved media directly into Radarr and Sonarr with your predefined quality profiles and root folders.

DB-Backed Persistance

All settings, logs, and recommendations are safely stored in a local SQLite database that persists across container rebuilds.

TMDb Dual-Engine

Combines smart AI reasoning with TMDb's massive dataset to ensure perfect metadata pairing (posters, genres, IDs).

Privacy First

100% self-hosted. Your watch history never leaves your network unless you explicitly enable the optional AI module.

Gorgeous Interface

A modern, responsive, dark-mode-first dashboard built with Next.js and Tailwind.

Install in Minutes

Deployed safely via Docker. Configured seamlessly via UI.

1
Create Compose File

Create a docker-compose.yml file on your server.

services:
  recomendarr:
    image: dheerajr00/recomendarr:latest
    container_name: recomendarr
    ports:
      - "3000:3000"
    volumes:
      - recomendarr-data:/app/data
    restart: unless-stopped

volumes:
  recomendarr-data:

2
Start & Setup

Run the compose command, then open your browser to complete the incredibly simple UI Setup Wizard.

docker-compose up -d

Open http://localhost:3000 in your browser.

Prefer standard Docker run?

docker run -d \
  --name recomendarr \
  -p 3000:3000 \
  -v recomendarr-data:/app/data \
  --restart unless-stopped \
  dheerajr00/recomendarr:latest
